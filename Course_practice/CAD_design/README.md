## Fusion360简介  

Fusion 360 是一款基于远程服务的三维建模、CAD、CAM、CAE 和 PCB 软件平台，主要用于产品设计和制造。以所需方式对产品进行设计和工程设计，在符合外形、结构和功能规格的同时确保美观。界面简介，软件体量小，支持线上共同协作。
可他为什么一定要安装在C盘。
