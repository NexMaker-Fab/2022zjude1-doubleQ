# 网站搭建

- 使用doscify搭建 下载 node.js
- 命令行键入 npm i docsify-cli -g 安装 docsify-cli 工具
- 在index.html文件中加入lodaSidebar: true 开启侧边栏
- 新建_sidebar.md文件
- 键入`<!-- docs/_sidebar.md -->`
- 键入`*[首页](zh-cn/)`
- 在index.html文件中加入lodaNavbar: true 开启导航栏
- 新建_navbar.md文件
- 键入`*[操作指南](guide)`
